# Lori

> **The AI coding workspace that grows with your team of agents.**

**AI models change. Coding agents change. Your workspace shouldn't.**

Lori is a vendor neutral workspace for AI coding that brings **Claude Code, Codex, OpenCode, Gemini CLI**, and future AI coding agents into one unified experience.

Manage your sessions, MCPs, providers, skills, and workspace from one place. Configure once, switch between agents instantly, and stay independent of any single AI vendor.

## Why Lori?

The AI coding ecosystem is moving fast.

Every new coding agent introduces its own configuration, MCP setup, provider management, skills, and workflow. As developers adopt multiple tools, their environment becomes fragmented and difficult to maintain.

Lori solves this by providing a single workspace that sits above every coding agent.

Instead of rebuilding your setup every time a better model or agent arrives, you keep one consistent workflow while using the tools that work best for each task.

## Features

* Unified workspace for multiple AI coding agents
* Shared MCP server management across supported agents
* Shared LLM provider configuration
* Shared skills and reusable workflows
* Seamless session management and agent switching
* Native terminal experience with multiple concurrent sessions
* Local first architecture with your data stored on your machine
* Automatic in app updates
* Designed to support new coding agents as they emerge

## Install (macOS)

1. Download the latest `.dmg` from **Latest Release**
2. Drag **Lori** into **Applications**
3. Launch Lori

If macOS blocks the first launch:

* Right click **Lori** and choose **Open**
* Or allow it from **System Settings → Privacy & Security**

Requires **macOS 10.15+** on **Apple Silicon**.

## Philosophy

Lori is not another AI coding agent.

It is the workspace that lets you use the best AI coding agent for every task without rebuilding your environment.

As models improve and new coding agents emerge, your workflow stays exactly the same.
